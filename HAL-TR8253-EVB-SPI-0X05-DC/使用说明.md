# EtherCAT 从站协议栈 — 业务工程师使用说明

## 一句话概括

你只需要在 **Application.c** 的两个函数里写业务逻辑，其他全部自动处理。

---

## 1. 你需要关心的文件（只有4个）

| 文件 | 作用 | 你需要动？ |
|---|---|---|
| `ECAT/ecat_api.h` | API 头文件，所有能调用的函数都在这里 | 只读 |
| `HARDWARE/Application.h` | GPIO 宏定义 + 回调声明 | 加 GPIO 时改 |
| `HARDWARE/Application.c` | **业务代码写在这里** | **主要编辑** |
| `Core/Src/main.c` | 初始化入口，已写好，一般不动 | 基本不动 |

## 2. 你不需要关心的文件（不要动）

```
SSC/          ← SSC Tool 生成的核心协议栈，改了会被覆盖
ECAT/ecat_hw_if.h
ECAT/ecat_api.c
HARDWARE/App_Flash.c/h  ← Flash 存储，自动调用
```

---

## 3. 快速上手 — main.c 怎么写

```c
#include "ecat_api.h"
#include "Application.h"

int main(void) {
    HAL_Init();
    SystemClock_Config();
    MX_GPIO_Init();
    MX_SPI1_Init();
    MX_TIM3_Init();

    // 1. 注册业务回调
    ECAT_RegisterPeriodicTask(App_Periodic_Task);
    ECAT_RegisterSafeOutput(App_Safe_Output_State);

    // 2. 业务外设初始化 (I2C/ADC 等)
    App_Hardware_Init();

    // 3. 启动 EtherCAT 协议栈 (自动从 Flash 恢复参数)
    ECAT_Stack_Init();

    // 4. 主循环
    while (1) {
        ECAT_Stack_MainLoop();  // 驱动协议栈 + 参数变更自动存 Flash
    }
}
```

## 4. 核心 API 参考

### 4.1 PDO 数据读写（在 App_Periodic_Task 里调用）

```c
// 单通道读写 (channel 范围: 1 ~ ECAT_DI_COUNT)
uint8_t ECAT_ReadDI(uint8_t channel);          // 返回 0 或 1
void    ECAT_WriteDO(uint8_t channel, uint8_t value);  // value: 0 或 1

// 整组读写 (16 路一次操作)
uint16_t ECAT_ReadDI_All(void);    // 每位对应一路 DI
void     ECAT_WriteDO_All(uint16_t value);
```

### 4.2 状态查询

```c
uint8_t ECAT_IsOpRunning(void);   // 返回 1 表示在 OP 状态且输出有效
uint8_t ECAT_GetAlState(void);    // 当前 AL 状态码
```

### 4.3 参数持久化（一般不需要手动调）

```c
void     ECAT_SetParamDirty(void);  // 标记参数已改，下次 MainLoop 自动存
uint32_t ECAT_GetParamSize(void);   // 参数块大小（字节）
void*    ECAT_GetParamPtr(void);    // 参数块指针
```

## 5. 写业务代码 — Application.c 模板

```c
#include "Application.h"

// 周期性自动调用（仅在 OP 状态执行）
void App_Periodic_Task(void)
{
    // 示例1: 读取 DI1 控制 DO1
    ECAT_WriteDO(1, ECAT_ReadDI(1));

    // 示例2: 读取传感器，写入 DO
    // uint8_t sensor = MySensor_Read();
    // ECAT_WriteDO(2, sensor > THRESHOLD ? 1 : 0);
}

// 断线/退出OP时自动调用，切断所有输出
void App_Safe_Output_State(void)
{
    ECAT_WriteDO_All(0x0000);   // 全部输出设为 0
    // MyMotor_Stop();          // 电机急停
}

// 外设初始化（在 ECAT_Stack_Init 之前调用）
void App_Hardware_Init(void)
{
    // My_I2C_Init();
    // My_ADC_Init();
}
```

**重要：** 不要在 `App_Periodic_Task()` 里做耗时操作（如 `HAL_Delay`），会阻塞 EtherCAT 通信。

## 6. Application.h — GPIO 宏定义

当硬件引脚变更时，只需改这里的宏，业务代码不受影响：

```c
// DO 输出引脚
#define DOCH0    PDout(14)
#define DOCH1    PDout(15)
// ... 按实际接线添加

// DI 输入引脚
#define DICH0()   PGin(15)
#define DICH1()   PBin(3)
// ... 按实际接线添加
```

## 7. SSC Tool 重新生成后的操作

当你用 SSC Tool 修改了 PDO 配置并重新生成代码后，SSC 目录下几乎所有文件都会被覆盖。**你需要补 4 个文件：**

| 文件 | 操作 |
|---|---|
| **SSC/TR8253Evb.h** | 末尾加 `#include "ecat_hw_if.h"`（在 `#undef PROTO` 之后） |
| **SSC/TR8253Evb.c** | 删 `#include "Application.h"`，加 `#include "ecat_api.h"` + HAL 头文件 |
| **SSC/SSC-Device.c** | 4 个函数里加函数指针调用（见附录） |
| **SSC/SSC-Device.h** | 末尾加 4 行 `extern` 函数指针声明（见附录） |
| **ECAT/ecat_api.h** | 如果对象字典变量名变了，更新 `ECAT_DI_VAR` / `ECAT_DO_VAR` 宏 |

---

## 8. 添加新的对象字典（如声纳数据、电机转速）

**第1步：** 在 SSC Tool 中添加 TxPDO / RxPDO 对象字典条目，生成代码。

**第2步：** 执行第7节的补丁操作。

**第3步：** 在 [ecat_api.h](ECAT/ecat_api.h) 中添加新的映射宏：
```c
#define ECAT_SONAR_VAR   SonarSensor0x6001   // 新增
```

**第4步：** 在 [ecat_api.c](ECAT/ecat_api.c) 中扩展 PDO 映射函数，或新增读写函数。

**第5步：** 在 `App_Periodic_Task()` 中调用新的读写函数。

> **简单场景**（只是多了几个 DI/DO 通道）：只需改 `ECAT_DI_COUNT` / `ECAT_DO_COUNT` 宏。
> **复杂场景**（全新数据类型）：找协议栈维护者（你）扩展 ecat_api.c 的映射函数。

## 附录 — SSC Tool 再生补丁详情

### A. SSC/TR8253Evb.h

在文件末尾 `#undef PROTO` 之后、`/** @}*/` 之前，加入：
```c
#include "ecat_hw_if.h"
```

### B. SSC/TR8253Evb.c

将 `#include "Application.h"` 替换为：
```c
#include "ecat_api.h"
#include "stm32f4xx_hal.h"
#include "spi.h"
#include "tim.h"
```

### C. SSC/SSC-Device.h

在文件末尾 `#undef PROTO` 之后、`/** @}*/` 之前，加入：
```c
extern void (*g_pfnPeriodicTask)(void);
extern void (*g_pfnSafeOutput)(void);
extern void (*g_pfnTxPdoMapping)(UINT16* pData);
extern void (*g_pfnRxPdoMapping)(UINT16* pData);
```

### D. SSC/SSC-Device.c

4 处函数替换：

| 函数 | 替换为 |
|---|---|
| `APPL_StopOutputHandler` | 函数体开头加 `if (g_pfnSafeOutput) g_pfnSafeOutput();` |
| `APPL_InputMapping` | 函数体改为 `if (g_pfnTxPdoMapping) g_pfnTxPdoMapping(pData);` |
| `APPL_OutputMapping` | 函数体改为 `if (g_pfnRxPdoMapping) g_pfnRxPdoMapping(pData);` |
| `APPL_Application` | 函数体改为 `if (bEcatOutputUpdateRunning && g_pfnPeriodicTask) g_pfnPeriodicTask();` |
