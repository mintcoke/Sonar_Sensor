EtherCAT 从站 — PLC 测试工程师使用说明


一、你要做的事
1. 导入 ESI 描述文件 (TRI_SPI_DC.xml)
2. 在 PLC 工程里把 PDO 变量映射到软元件 (W/D/M 寄存器)
3. 在线监视观察数据


二、导入 ESI 文件
文件位置: 工程目录下的 TRI_SPI_DC.xml
KV-Studio 操作步骤:
  1. 打开 KV-Studio → 菜单 工具 → EtherCAT 配置器
  2. 右键 从站设备 → 添加从站 → 浏览 ESI 文件
  3. 选择 TRI_SPI_DC.xml
  4. 从站识别为 SSC-Device (厂商 Triductor)
如果已有从站，右键从站 → 更新 ESI → 选择新的 TRI_SPI_DC.xml


三、PDO 映射速查表

3.1 从站发给 PLC (TxPDO / 输入) — PLC 侧叫"输入"，对应固件的 DI(n)

  PDO 位置          | 建议软元件 | 固件变量           | 含义
  --------------------------------------------------------------------
  0x1A00 SubIdx 1    | W100      | DI(T_STATE)        | 设备状态
  0x1A00 SubIdx 2    | W101      | DI(T_ALARM)        | 报警码
  0x1A00 SubIdx 3    | W102      | DI(T_VERSION)      | 固件版本
  0x1A00 SubIdx 4    | W103      | DI(T_DISTANCE)     | 声纳距离 (mm)
  0x1A00 SubIdx 5    | W104      | 预留                | -
  ...                | ...       | ...                | ...
  0x1A00 SubIdx 128  | W227      | 预留                | -
  0x1A01 SubIdx 1    | W228      | 预留                | -
  ...                | ...       | 共 510 个 UINT16    | ...
  0x1A01 SubIdx 127  | W609      | 预留                | -

核心关注: W103 = 声纳距离 mm，实时更新。

3.2 PLC 发给从站 (RxPDO / 输出) — PLC 侧叫"输出"，对应固件的 DO(n)

  PDO 位置          | 建议软元件 | 固件变量           | 含义
  --------------------------------------------------------------------
  0x1600 SubIdx 1    | W0        | DO(R_LED)          | LED 控制字
  0x1600 SubIdx 2    | W1        | 预留                | -
  ...                | ...       | 共 510 个 UINT16    | ...

核心关注: W0 = LED 控制，写 0x0001 亮 LED0，写 0x0002 亮 LED1...


四、怎么在 KV-Studio 映射到软元件
1. 双击 EtherCAT 从站 → PDO 映射 标签页
2. 左侧 TxPDO 列表, 选中一行 → 右侧 映射目标 选 "W 寄存器" → 输入起始地址 (如 W100)
3. 同样把 RxPDO 映射到输出区 (如 W0)
4. 下载到 PLC


五、在线监视
1. PLC 在线连接 (USB 或以太网)
2. KV-Studio → 监视模式 (F3)
3. 打开 软元件监视窗口, 输入 W103 看声纳距离
4. 在声纳前方放障碍物, W103 数值会变 (单位 mm)

快速验证数据通不通:
  声纳距离: 监视 W103, 前方无障碍时应为大值 (>8000) 或 0
  放个 1m 障碍物应看到约 1000
  LED 控制: 软元件写入 W0 = 0x0003 → 板子上 LED0 和 LED1 亮


六、新增 PDO 变量后的操作
当固件工程师在 ecat_api.h 里加了新变量 (如 T_TEMP 对应 DI(5))，你需要:
1. 确认 DI(5) 在 TxPDO 里的位置 (5 从 0 开始 = 0x1A00 SubIdx 6)
2. 如果对应的寄存器还没映射，加上去
3. 不需要重新导入 ESI (除非 PDO 结构变了)


七、常见问题

Q: PLC 读到数值永远是 0
A: 1. 从站进 OP 了吗？看 PLC 里从站状态应为 OP
   2. 固件工程师确认 DI(T_DISTANCE) = Sonar_Poll() 确实在跑

Q: ESI 导入失败
A: 确认 XML 文件编码是 UTF-8，没有被记事本改过

Q: 值看起来不对
A: 确认接线正确：声纳 RX→PC6，声纳 TX→PC7，GND↔GND
